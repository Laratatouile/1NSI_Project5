Project 5 de NSI

I) Qu'est ce que le jeu

1) Qu'est de que le jeu

Dans ce simple jeu vous devez atteindre un drapeau déposé sur la carte

2) Comment ça fonctionne ?

Vous pouvez vous deplacer sur la carte à l'aide des touches ZQSD.

3) Evitez les monstres

Des monstres sont arrivés sur la carte a vous des les éviter

4) Les obstacles

Il y a deux types d'obstacles:

  • Les cases eau qui vous font ralentir
  • Les cases lave qui vous transforment en barquette de frites.
  • La case du drapeau.

II) Explication breve du code

1) Le fichier main

C'est celui qui va executer les differentes instances et il va aussi initailiser les viriables de base.

2) Les fichiers de fonctions

Ce sont les fichiers qui s'occupent de taches comme par exemple le chiffrement des sauvgardes, les ouvertures de fichiers et la récupération des touches associées aux actions.

3) Les fichiers d'objects

Ce sont les fichiers qui s'occupent de calculer objects et des decors. Il y en a un spécialisé dans l'affichage pyxel avec plein de parametres.
